# User Guide — AI-Fit

## Welcome

AI-Fit helps you find the best AI model for every part of your application. Whether you're building a chatbot, a code assistant, or an image generator, we match your project to the right provider and model.

## Getting Started

1. **Launch the app** and tap **Get Started**.
2. **Describe your project** in a sentence or two.
3. **Select use-cases** that apply (e.g., Chat, Code-assist, Image generation).
4. **Set constraints**: budget, latency, compliance needs, and preferred providers.
5. **Review your answers** and submit.

## Understanding Recommendations

Each result card shows:
- **Model name** and provider badge
- **Why it was chosen** (cost, latency, capabilities)
- **Estimated monthly cost**
- **Copy-ready code snippet** in TypeScript

Tap the **Copy** button to grab the snippet, or tap **Save** to store the configuration for later.

## Exporting Configurations

On the Results screen, tap **Export Config** to download a JSON or YAML file. You can import this into your CI/CD pipeline or share it with your team.

## Managing Settings

Navigate to the **Settings** tab to:
- Add or update your API keys (stored securely)
- Toggle dark mode
- View and manage saved projects

## Analytics

The **Analytics** tab shows how many recommendations you've generated and summarizes estimated costs across your saved projects.

## Need Help?

If a recommendation seems off, try relaxing your constraints (e.g., increase budget or latency tolerance) and submit again.
